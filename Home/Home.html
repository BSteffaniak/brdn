Nova

January 2014 - Present

public class HelloWorld
{
    [EntryPoint]
    public static sayHello()
    {
        // No semi-colons needed!
        Console.writeLine("Hello, world!")
    } 
}

Nova is a general purpose programming language that I am developing. The syntax is Java and C# inspired. To the left you see a basic hello world program. There are two key differences from java and C#:

  1. You can give your "main" function any name. All that is required is an [EntryPoint] annotation for the compiler to know to use that function to start the program.
  2. No semi-colons are needed to end a statement.
public class ProcessLists
{
    [EntryPoint]
    public static processLists()
    {
        var list = [5, 10, 15, -16, 17]

        var evens = list.filter(x -> x % 2 == 0)
        var odds = list - evens
        var subRange = list.filter(x -> 10 < x < 16)
        var squares = list.map(x -> Math.pow(2))
        var absoluteValues = list.map(Math.abs) // Pass a function
    } 
}

Nova's List data structure makes processing lists of data easy. With the use of lambda expressions, the code to perform operations can be concise.

Like many other modern languages, Nova lets you use lambda expressions and functions as first class values.

Notice that Nova is smart enough to infer the type of the variables that were defined with the use of the keyword "var".

Nova is, and has always been, developed in my free time. The reason I am making it comes down to the fact that I wanted a cleaner syntax to work with.

The Digging GameThe Digging Game

May 2012 - Jun 2013

The Digging Game is my most recent attempt at making a video game. There have been many iterations of the game ranging from when I was just learning GUI development. The game started off built with the Java Swing framework. The game is currently built in Java with OpenGL used to render the graphics.

The game's terrain generation was infinite in all (2 dimensional) directions.

The game has a functioning day/night cycle. During the day the environment is much more visible than during the night (unless a light source, such as a torch, is present). There is a aesthetically appealing transition from day to night (and vice versa) where pretty colors like pink can be seen.
